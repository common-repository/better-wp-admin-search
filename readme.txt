Add essential search functionality to your WP Admin.

== Description ==

Better WP-Admin Search adds global search functionality to your WP-Admin.
- Search in WordPress native post types (Posts, Pages, Media, Navigation Menu Items, Custom CSS, Global Styles)
- Search in plugin custom post types
- View and edit search results

== Frequently Asked Questions ==

= How do I install Better WP-Admin Search =

Upload the directory `/better-wp-admin-search/` to your WP plugins directory and activate the plugin.
